QUICK START: DEPLOY IN 5 MINUTES
===============================================

TLDR: Follow these 10 commands:

1. CREATE GITHUB REPO
   https://github.com/new
   Name: SocialApp
   Public or Private (both work)
   Click "Create repository"

2. CLONE REPO LOCALLY
   git clone https://github.com/YOUR_USERNAME/SocialApp.git
   cd SocialApp

3. COPY YOUR FILES
   Copy into SocialApp folder:
   - SocialAppServer.java
   - NetworkClient.java
   - Procfile (already created)
   - Dockerfile (already created)
   - lib/ folder with all JAR files

4. PUSH TO GITHUB
   git add .
   git commit -m "Initial commit: SocialApp server"
   git push -u origin main

5. DEPLOY ON RENDER
   a) Go to https://render.com (sign up free)

UPDATING SERVER CODE
====================

To update your server after deployment:

1. Make changes to SocialAppServer.java
2. Commit and push:
   git add .
   git commit -m "Fixed bug"
   git push

3. Render auto-deploys in 1-2 minutes!
   (Check deployment logs in Render dashboard)


COMMON JAR FILES NEEDED
=======================

Make sure your lib/ folder contains:
- gson-2.x.jar (JSON)
- webcam-capture-0.3.12.jar (camera)
- Any other dependencies your project uses

Find your dependencies in your build/lib or check your original compilation setup.
