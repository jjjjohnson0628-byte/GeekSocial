## â“ TROUBLESHOOTING

### "Java is not installed" error
**Solution:** Download and install Java from oracle.com

### "Port 8080 already in use" warning
**Solution:** 
- Another program is using port 8080
- Close other applications or change the port in the code

### "Server not available" message in app
**Solution:**
- Make sure StartServer.bat is still running
- Restart the server by closing and re-running it

### Online Mode won't enable
**Solution:**
- Check that Server window is open
- Try: Menu > Mode > Enable Online Mode > Toggle ON

### Posts not syncing between users
**Solution:**
- Restart both apps
- Make sure both have Online Mode enabled
- Check if server window shows any errors
