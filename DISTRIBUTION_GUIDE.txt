# SocialApp Distribution Guide

## üéØ For Users Who Want to Share This App

This guide explains how to make your SocialApp easily shareable with others.

---

## üì¶ METHOD 1: Simple ZIP Package (RECOMMENDED)

### Create a Shareable Package:

1. **Windows Users**:
   - Double-click `Package.bat` in the SocialApp folder
   - Wait for it to complete
   - A file named `SocialApp_[date].zip` is created
   - Send this ZIP file to others

2. **macOS/Linux Users**:
   ```bash
   zip -r SocialApp.zip SocialApp/
   ```

### Recipients Install:

1. Extract the ZIP file
2. Install Java if needed (see below)
3. Double-click `Launch.vbs` (Windows) or run `bash run.sh` (macOS/Linux)

---

## ‚òï JAVA INSTALLATION FOR RECIPIENTS

### Windows:

1. Go to: https://www.oracle.com/java/technologies/downloads/
2. Click "Download JDK" (latest version, Windows x64)
3. Run the installer
4. Follow the wizard (keep defaults)
5. Restart computer
6. Done!

**Alternative (OpenJDK):**
- Go to: https://adoptopenjdk.net/
- Download and install

### macOS:

1. Open Terminal
2. Install Homebrew if needed:
   ```bash
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   ```
3. Install Java:
   ```bash
   brew install openjdk
   ```
4. Done!

### Linux (Ubuntu/Debian):

```bash
sudo apt update
sudo apt install openjdk-11-jdk
```

### Linux (Fedora/RHEL):

```bash
sudo dnf install java-11-openjdk
```

## üêõ IF RECIPIENT HAS ISSUES

### "App won't start"
- Make sure Java is installed and working
- Try running `SystemCheck.bat` for diagnostics
- Restart computer

### "Webcam doesn't work"
- Grant permissions if prompted
- Check only one app is using webcam
- Restart the app

### "Can't find files"
- Ensure all files extracted
- Don't move files individually
- Keep folder structure intact

## üéØ PRO TIPS FOR DISTRIBUTION

**Small ZIP**: Only copy essential files (no media/ folder)
```
- Keep: .bat files, .vbs file, .jar files, javafx-sdk
- Skip: media/, temp/, old data
- Result: 50-100MB ZIP file
```

**Data Migration**: If you want to share posts/users
```
- Include data/users.json and data/posts.json
- Recipients will see your posts/users when they extract
```

**Read-Only Sharing**: For demos
```
- Don't include data/ folder
- Fresh start for each person who runs it
```

## ü§î COMMON QUESTIONS

**Q: Can they modify the app?**
A: Yes, it's open source! SocialApp.java has all the source.

**Q: Will their data sync with mine?**
A: No, each computer has separate data. To share: manually copy data/users.json and data/posts.json

**Q: What if they find a bug?**
A: You can both edit SocialApp.java and rebuild! Run Build.bat to compile.

**Q: Can they run it on their phone?**
A: Not yet. Java/JavaFX isn't typical on mobile. Android/iOS version would need rewrite.
