## ğŸ› TROUBLESHOOTING

### App won't start
1. Check Java is installed: `java -version`
2. If not, download and install Java
3. Make sure GeekSocial.jar exists
4. If missing, run `Build.bat`

### Webcam not working
- Check other apps aren't using it
- Try restarting
- Grant permissions if prompted

### Can't find files
- Stay in the SocialApp folder
- Don't move files individually
